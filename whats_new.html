Acoustics 2.0 - What's New [beta]

Welcome to the new Acoustics! The team has spent a number of months working on new features and a brand new interface for this release. This document provides an overview of what is new, what has changed, and what has been removed from Acoustics since our original release.

Interface Features

Acoustics 2 brings a brand new, HTML5/CSS3 interface with a whole new look and feel.
  • Built on JQuery
  • Designed to work with tablets
  • Uses modern browser features from CSS3 and HTML5
  • Integrated album art support
  • New fullscreen mode
  • Quick search results and autocompletion

Queue Features

This release includes a number of usability enhancements for the queue and playlists.
  • Reorder the queue and playlists by dragging and dropping songs
  • Easily identify songs you have voted for
  • See how many people voted for each song in the queue

Album Search

Acoustics 2 comes with an integrated album search page.
  • Search for albums by title or artist
  • Integrated coverflow-like search results

More Statistics

Acoustics now keeps track of the top voted songs and artists.
